Digital Music Stand

To install, type "install.sh"

To run, type "~/start_dms.sh". This will start the app and launch a browser
to display it.

Then follow the instructions to upload scores and create a playlist.

View the scores by clicking the "play" button (or if you restart DMS,
you'll be sent straight to the first page)

To advance a page, hit any key on the right-hand-side of the keyboard.
To go back a page, hit any key on the left-hand-side of the keyboard.
